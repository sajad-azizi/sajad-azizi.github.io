Research

Current activity

Attosecond photoionization time delays in molecules; theory and large-scale numerics. I build and optimize HPC codes (MPI/OpenMP/GPU/SYCL/Kokkos) and develop models to interpret ultrafast strong-field phenomena.

Selected projects

  • SPION — electron–molecule scattering (C++, OpenMP, MPI, SYCL). Continuum Hartree–Fock and DFT pipeline; Numerov scattering states; GPU-SYCL optimization. Source code on request.
  • Coupled-channel Schrödinger equation (C++, Eigen, OpenMP, GSL). Precise bound-state calculations in 2D/3D for model molecular potentials. Source code on request.
  • Time-dependent Schrödinger equation (C++, MPI, OpenMP, FFTW). Full 3D TDSE via high-order Numerov; hybrid MPI–OpenMP parallelism; 10× speedups for arbitrary laser pulses. Source code on request.
  • Machine Learning for photoelectron spectra (Python, TensorFlow/Keras/Sklearn). Correlations between two‑photon and net zero‑photon spectra. GitHub repo.
