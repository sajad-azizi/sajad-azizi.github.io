SPION — electron–molecule scattering & model potentials

Overview

SPION is an ab initio electron–molecule scattering package written in modern C++ with OpenMP, MPI, and SYCL acceleration. It builds a pipeline from molecular electronic structure to high-accuracy continuum (scattering) states. In addition to ab initio targets, SPION solves arbitrary 3D model potentials for both bound and continuum properties.


“Its standout feature is basis-set independence with excellent performance in high-energy applications.”


Capabilities

  • Electron–molecule scattering (ab initio): Continuum Hartree–Fock (static-exchange) with DFT pre-processing for orbital energies and coefficients; high-accuracy continuum wavefunctions suitable for phase shifts and cross-section workflows.
  • Model-potential mode (arbitrary 3D V(r)): accepts analytic or gridded user-supplied potentials and computes:
    • Ground state energy and wavefunction.
    • Excited discrete states (energies and eigenfunctions, when present).
    • Scattering states at user-specified energies E > 0.
  • Dipole matrix elements & overlaps between initial (e.g., ground) and final states, e.g. ⟨ψE(+)|μ|ψ0⟩ and ⟨ψE(+)|ψn⟩.

Pipeline

  1. DFT pre-processing → molecular orbitals & energies.
  2. Continuum Hartree–Fock (static-exchange) setup.
  3. Numerov integration for scattering channels and robust bound-state solvers.
  4. Post-processing → observables, phase shifts, dipole matrix elements, overlaps.

Numerics & performance

  • Numerov integration for partial-wave radial equations in scattering workflows.
  • Bound-state eigenvalue search, normalization, and orthogonality checks.
  • GPU offloading via SYCL; multicore parallelism with OpenMP; distributed runs via MPI.

Inputs & outputs

  • Inputs
    • Ab initio mode: molecular geometry + electronic-structure data (orbitals/energies).
    • Model-potential mode: V(r) as a callable/analytic form or a 3D grid (uniform or rectilinear).
  • Outputs
    • Bound spectra {En} and eigenfunctions {ψn(r)}.
    • Scattering solutions ψE(+)(r) at requested energies.
    • Dipole matrix elements and state overlaps for transition/ionization analyses.

Status & availability

Actively developed. Source code available on request. Benchmarks, examples, and documentation will be added here.
